단순 연결 리스트(Linked List) 자료구조를 활용한 도서 관리 프로그램

----------------------------------------
■ 프로젝트 개요
----------------------------------------
이 프로그램은 파이썬으로 구현된 단순 연결 리스트(Linked List) 구조를 기반으로
도서를 추가, 삭제, 조회, 전체 목록 출력할 수 있는 도서 관리 시스템입니다.

자료구조 학습용 예제로, 
Node / LinkedList / Book / BookManagement 클래스로 구성되어 있습니다.


----------------------------------------
■ 실행 방법
----------------------------------------
python book_management.py

실행 후 아래 메뉴가 표시됩니다.

=== 도서 관리 프로그램 ===
1. 도서 추가
2. 도서 삭제 (책 제목으로 삭제)
3. 도서 조회 (책 제목으로 조회)
4. 전체 도서 목록 출력
5. 종료


----------------------------------------
■ 주요 클래스 구조
----------------------------------------

[1] Node 클래스
- 연결 리스트의 기본 단위
- 각 노드는 data와 link(다음 노드)로 구성

[2] LinkedList 클래스
- 연결 리스트의 삽입, 삭제, 탐색 기능을 구현
- 주요 메서드
  • insert(elem, pos): 지정 위치에 노드 삽입
  • delete(pos): 지정 위치의 노드 삭제
  • find_by_title(title): 제목으로 노드 존재 여부 확인
  • find_pos_by_title(title): 제목이 저장된 노드의 위치 반환
  • display(): 전체 리스트 출력

[3] Book 클래스
- 단일 도서 정보를 저장하는 클래스
- 단순 필드 구조

[4] BookManagement 클래스
- 전체 도서 목록 관리 및 CRUD 기능 수행
- 사용자 입력을 통해 메뉴 기반으로 동작

저장 구조:
[ Book 객체 ] → [ "제목_total" ] → [ Book 객체 ] → [ "제목_total" ] ...

주요 메서드
  • add_book(): 도서 추가
  • remove_book(): 제목으로 도서 삭제
  • search_book(): 제목으로 도서 조회
  • display_books(): 전체 도서 목록 출력
  • run(): 메뉴 루프 실행

■ 실행 예시
----------------------------------------

=== 도서 관리 프로그램 ===
1. 도서 추가
2. 도서 삭제 (책 제목으로 삭제)
3. 도서 조회 (책 제목으로 조회)
4. 전체 도서 목록 출력
5. 종료

메뉴를 선택하세요: 1
책 번호 입력 : 101
책 제목 입력 : 알고리즘의정석
저자 입력 : 김철수
출판 연도 입력 : 2020
도서 '알고리즘의정석'가 추가되었습니다.

메뉴를 선택하세요: 4
현재 등록된 도서 목록:
[책 번호: 101, 제목: 알고리즘의정석, 저자: 김철수, 출판 연도: 2020]

메뉴를 선택하세요: 2
삭제할 책 제목 입력 : 알고리즘의정석
'알고리즘의정석' 도서가 삭제되었습니다.
